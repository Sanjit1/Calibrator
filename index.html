Thermistor Calculator

To calculate the temperature from the resistance of a thermistor, we need to use the Steinhart-hart equation.
`1/T = A + BlnR + ClnR^3`
Where T is temperature, R is resistance, and A,B,C are Steinhart coefficients, which vary from thermistor to thermistor. To find the temperature, we need to find the three Steinhart coeffients.
